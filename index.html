to-domain

Convert a string to a valid domain name equivalent

v1.0.1

Tweet

Installation

npm install --save to-domain

Usage

var toDomain = require('to-domain')

toDomain('    foo-bar/baz   yay') // => 'foo-bar-baz-yay'
